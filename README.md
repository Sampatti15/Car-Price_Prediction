# 🚗 Car Price Prediction System

A Machine Learning-based web application that predicts the selling price of a used car based on various features such as car age, fuel type, transmission, kilometers driven, ownership, and more.

The project utilizes **Linear Regression** and **Lasso Regression** algorithms to build predictive models and compares their performance to provide accurate price estimation.

---

## 📌 Features

- Predicts the estimated selling price of a used car.
- Data preprocessing and feature engineering.
- Model training using:
  - Linear Regression
  - Lasso Regression
- Model performance evaluation using multiple metrics.
- Interactive prediction interface.

---

## 🛠️ Tech Stack

- Python

- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Machine Learning Workflow

1. Data Collection
2. Data Pre Processing
3. Train-Test Split
4. Model Training
   - Linear Regression
   - Lasso Regression
5. Model Evaluation
6. Price Prediction

---

## 📈 Evaluation Metrics

- R Squared Error

---

## 📂 Dataset

The dataset contains information about used cars, including:

- Car Brand
- Year
- Selling Price 
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner


---

## 🎯 Objective

The primary objective of this project is to build an efficient machine learning model capable of accurately predicting used car prices, helping buyers and sellers make informed decisions.
