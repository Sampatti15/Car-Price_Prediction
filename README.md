# 🚗 Car Price Prediction System

A Machine Learning-based web application that predicts the selling price of a used car based on various features such as car age, fuel type, transmission, kilometers driven, ownership, and more.

The project utilizes **Linear Regression** and **Lasso Regression** algorithms to build predictive models and compares their performance to provide accurate price estimation.

---

## 📌 Features

- Predicts the estimated selling price of a used car.
- Data preprocessing and feature engineering.
- Model training using:
  - Linear Regression
  - Lasso Regression
- Model performance evaluation using multiple metrics.
- Interactive prediction interface.
